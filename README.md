tmst
====

Simple timesheet software.

Usage
-----

```console
$ tmst in foo
$ tmst current
foo
$ # do some work...
$ tmst out
$ tmst list
1234-56-78
==========

foo - 0.15h

Total: 0.15h
```

Installation
------------

Grab a binary from [Releases](https://github.com/remexre/tmst/releases) or install one from crates.io with `cargo install tmst`.
